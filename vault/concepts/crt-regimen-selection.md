## 背景・位置づけ
局所進行（III期）・切除不能NSCLCに対する根治的化学放射線療法（CRT）には、化学療法と放射線療法を同時に行う同時併用（concurrent CRT, cCRT）と、時間をずらして行う逐次併用（sequential CRT, sCRT）の2方式がある。両者を直接比較した複数のランダム化比較試験を統合した個別患者データメタアナリシス（Aupérin 2010）が、現在cCRTが標準治療とされる根拠となっている。

## メタアナリシスの概要
- 対象：データ提供のあった6件のランダム化比較試験、1205例（全ランダム化患者の92%）
- 対象患者：遠隔転移のない未切除NSCLC、根治目的の胸部放射線療法を予定、過去の放射線療法・化学療法歴なし
- 主要評価項目：全生存期間（OS）

## 結果

| 項目 | cCRT vs sCRT | 備考 |
|---|---|---|
| OS | HR 0.84（95%CI 0.74-0.95）, p=0.004 | cCRTが有意に良好 |
| 3年生存率 | 23.8% vs 18.1%（絶対差+5.7%） | |
| 5年生存率 | 15.1% vs 10.6%（絶対差+4.5%） | |
| 局所制御 | HR 0.77（95%CI 0.62-0.95）, p=0.01 | OSベネフィットの主因 |
| 遠隔転移 | HR 1.04（95%CI 0.86-1.25）, p=0.69 | 有意差なし |
| 急性食道毒性（Grade3-4） | 18% vs 4%、RR 4.9（95%CI 3.1-7.8）, p<0.001 | cCRTで有意に増加 |
| 急性肺毒性 | RR 0.69（95%CI 0.42-1.12）, p=0.13 | 有意差なし |

## 結論・臨床的意義
cCRTはsCRTと比較して、局所制御の改善を主因としてOSを有意に改善する。一方で急性食道毒性は明確に増加するため、患者の全身状態・治療忍容性を考慮したレジメン選択が必要となる。

## PACIFICとの関係
PACIFIC試験（durvalumab地固め療法）はcCRT後の患者のみを組み入れ対象としている。sCRT後の患者に対するdurvalumab地固めの効果は、PACIFIC-5試験・PACIFIC-6試験で別途検証されている（本サイトでは今後追加予定）。
