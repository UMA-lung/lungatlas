肺の解剖学

外科手術・気管支鏡・画像診断に必要な肺の基本解剖。肺葉・肺区域・気管支分岐・肺門構造・リンパ節分類を整理する。リンパ節分類はIASLC Staging Cards 9th Edition 2024および肺がん取り扱い規約 第9版に準拠。

肺の全体像

葉と葉間裂

肺は左右に分かれ、右肺は3葉・左肺は2葉からなる。葉間裂により区分される。

右肺 3葉
  • 上葉（RUL）：水平裂の上方
  • 中葉（RML）：水平裂と斜裂の間
  • 下葉（RLL）：斜裂の後下方
葉間裂：水平裂（上葉–中葉）＋斜裂（中葉–下葉）
左肺 2葉
  • 上葉（LUL）：舌区（S4・S5）を含む
  • 下葉（LLL）：斜裂の後下方
葉間裂：斜裂のみ（水平裂は原則存在しない）

肺区域解剖（Bronchopulmonary Segments）

肺区域は各区域気管支（B）に対応する機能的単位。区域切除術では区域間平面に沿って切離するため、区域気管支・区域動脈・区域静脈の正確な把握が必須。

気管支分岐

気管（trachea）は隆起（carina）で左右の主気管支に分岐する。

右側気管支分岐
右主気管支（Right main bronchus）
├ 右上葉気管支
├ B1 肺尖区
├ B2 後上区
└ B3 前上区
└ 中間幹気管支（Bronchus intermedius）
├ 右中葉気管支
├ B4 外側区
└ B5 内側区
└ 右下葉気管支
├ B6 上区（背区）
├ B7 内側底区
├ B8 前底区
├ B9 外側底区
└ B10 後底区
左側気管支分岐
左主気管支（Left main bronchus）
├ 左上葉気管支
├ B1+2 肺尖後区
├ B3 前上区
└ 舌区気管支
├ B4 上舌区
└ B5 下舌区
└ 左下葉気管支
├ B6 上区（背区）
├ B8 前底区
├ B9 外側底区
└ B10 後底区

肺門構造

肺門は気管支・肺動脈・肺静脈・リンパ管・神経が出入りする部位。肺葉切除・区域切除では肺門での血管・気管支処理が基本となる。

右肺門

前方（腹側）から後方（背側）の順：

  1. 上肺静脈（Superior pulmonary vein）
  2. 肺動脈（Pulmonary artery）
  3. 右上葉気管支（Upper lobe bronchus）
  4. 中間幹気管支（Bronchus intermedius）
  5. 下肺静脈（Inferior pulmonary vein）
左肺門

前方から後方の順：

  1. 上肺静脈（Superior pulmonary vein）
  2. 肺動脈（Pulmonary artery）— 気管支の上方をアーチ状に走行
  3. 左上葉・左下葉気管支
  4. 下肺静脈（Inferior pulmonary vein）

リンパ節分類

肺癌のリンパ節転移評価にはIASLC（国際肺癌研究学会）のリンパ節地図が国際標準として使用される。N因子（N1/N2/N3）は手術適応・治療方針に直結する。

リンパ節分類（IASLC Staging Cards 9th Edition 2024 / 肺がん取り扱い規約 第9版）

以下の英語名称・境界定義はIASLC Staging Cards 9th Edition 2024に準拠。日本語名称は肺がん取り扱い規約 第9版（表1 CT読影基準）に対応。

# IASLC 名称 日本語名称（第9版） N因子 上縁 / 下縁（IASLC 9th Ed. 2024）
#1 — Low cervical, supraclavicular and sternal notch nodes
#1R / #1L Low cervical, supraclavicular & sternal notch 低頸部・鎖骨上窩・胸骨切痕リンパ節 N3 上縁：輪状軟骨下縁 / 下縁：両側鎖骨・正中では胸骨柄上縁。気管正中線で#1R・#1Lを区別
上部縦隔 — #2〜#4
#2R Upper paratracheal (R) 右上部気管傍リンパ節 N2（右肺）/ N3（左肺） 上縁：肺尖・胸膜頂（正中：胸骨柄上縁）/ 下縁：左腕頭静脈尾側縁と気管の交差部。気管左外側縁より右側
#2L Upper paratracheal (L) 左上部気管傍リンパ節 N2（左肺）/ N3（右肺） 上縁：肺尖・胸膜頂（正中：胸骨柄上縁）/ 下縁：大動脈弓上縁。気管左外側縁より左側
#3a Prevascular 前血管リンパ節 N2 上縁：胸膜頂 / 下縁：隆起レベル。胸骨後方・前縁：胸骨後面。後縁：右＝上大静脈前縁、左＝左総頸動脈
#3p Retrotracheal 気管後リンパ節 N2 上縁：胸膜頂 / 下縁：隆起レベル。気管後縁より後方
#4R Lower paratracheal (R) 右下部気管傍リンパ節 N2（右肺）/ N3（左肺） 上縁：左腕頭静脈尾側縁と気管の交差部 / 下縁：奇静脈下縁。右気管傍・気管前リンパ節を含む。気管左外側縁より右側
#4L Lower paratracheal (L) 左下部気管傍リンパ節 N2（左肺）/ N3（右肺） 上縁：大動脈弓上縁 / 下縁：左主肺動脈上縁。動脈管索内側のリンパ節を含む。気管左外側縁より左側
大動脈リンパ節 — #5〜#6（左肺原発に関連）
#5 Subaortic (AP window) 大動脈下リンパ節（大動脈肺動脈窓） N2（左肺）/ N3（右肺） 上縁：大動脈弓下縁 / 下縁：左主肺動脈上縁。動脈管索の外側（lateral）に位置
#6 Para-aortic (ascending aorta or phrenic) 傍大動脈リンパ節 N2（左肺）/ N3（右肺） 上縁：大動脈弓上縁に接する接線 / 下縁：大動脈弓下縁。上行大動脈・大動脈弓の前方・外側
下部縦隔 — #7〜#9
#7 Subcarinal 気管分岐下リンパ節 N2（両側） 上縁：隆起 / 下縁：左＝下葉支上縁、右＝中間幹下縁
#8R / #8L Para-esophageal (below carina) 傍食道リンパ節（隆起下より下方） N2（両側） 上縁：左＝下葉支上縁、右＝中間幹下縁 / 下縁：横隔膜。食道壁に接し正中線の左右に存在（#7を除く）
#9R / #9L Pulmonary ligament 肺靱帯リンパ節 N2（両側） 上縁：下肺静脈 / 下縁：横隔膜。肺靱帯内に位置
N1リンパ節 — #10〜#14
#10R / #10L Hilar 肺門リンパ節 N1 主気管支・肺門血管に隣接。肺静脈近位部・主肺動脈を含む。上縁：右＝奇静脈下縁、左＝肺動脈上縁 / 下縁：両側葉間領域
#11（#11s / #11i） Interlobar 葉間リンパ節 N1 葉気管支起点の間。右肺：#11s＝上葉支〜中間幹間、#11i＝中葉支〜下葉支間（任意のサブカテゴリ）
#12 Lobar 葉気管支周囲リンパ節 N1 葉気管支に隣接
#13 Segmental 区域気管支周囲リンパ節 N1 区域気管支に隣接
#14 Sub-segmental 亜区域気管支周囲リンパ節 N1 亜区域気管支に隣接
N因子の要点：#1（鎖骨上窩）はN3。#2・#4は同側N2 / 対側N3。#5・#6は左側原発N2 / 右側原発N3相当。#7〜#9は両側ともN2。#10〜#14はN1（同側）/ N3（対側）。
